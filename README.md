## Статус: `не закончен`

## Способ работы;

* Создаем структуру;
```
src/
    scripts/
    assets/
SFbuild.json
```

SFbuild.json - Конфигурация сборки 

* Запускаем сборку 
`sf build`
---
В той же директории

* Видим новую папку 
```
src/ 
build/ <-- вот 
SFbuild.json 
```
Там будет готовая игра;
