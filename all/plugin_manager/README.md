# Менеджер плагинов

Статическая библиотека для загрузки динамических библиотек.Также он инициализарует и предостовляет всем модулям функции из core.
В это модулей есть главный класс.

## `PluginManager`
Содержить в себе путь к файлу `modules.json` который хранит предназначение и необходимость каждого модуля.
### Пример:
```
{
  "modules":[
    {
      "id":"render",
      "path":"modules/render.dll",
      "status":"required"
    },
    {
      "id":"ui",
      "path":"modules/ui.dll",
      "status":"required"
    },
    {
      "id":"sound",
      "path":"modules/sound.dll",
      "status":"optional"
    }
 ]
}


```

## Задача
Задача класса `PluginManager` найти файл modules.json прочитать данные и начать загрузку всех плагинов по очереди.
