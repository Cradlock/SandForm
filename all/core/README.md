# Основной центр от которого все зависимы

Модуль который будут использовать все остальные,мозг игры.

---

В самом модуле есть несколько главных обьектов:



### class Engine 
> Основной двигатель 


### class Eventbus 
> Общение между модулями

### class ResourceManager
> Менеджер ресурсов

### class Time 
> Нужен для системы тиков

### class Config 
> Менеджер настроек (push model)

### class Logger 
> Система логирования

### namespace interfaces
> Контракты между модулями

- warning.log

- info.log

новая строка,а с ней и дата.
